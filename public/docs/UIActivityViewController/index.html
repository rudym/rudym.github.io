UIActivityViewController

Save, share, print and etc. Use it modally on phones and for iPad it is popover.

activityItems is the items you are throwing chosen event for. In the test code it is an image object.

let image = UIImage()

let mycontroller = UIActivityViewController(
	activityItems: [image], applicationActivities: nil)

self.presentViewController(mycontroller, animated: true,
	completion: nil)
